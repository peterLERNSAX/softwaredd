# Django-Backend

- developer: Peter Koban
- Framework: FastAPI [Docs](https://fastapi.tiangolo.com/)

Needed installation

```bash
    source .env/bin/activate
    pip install django
```

Start the webserver

```bash
    python manage.py runserver 8000
```
