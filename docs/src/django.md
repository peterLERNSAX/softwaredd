# Django-Backend

- developer: Peter Koban
- Framework: Django [Docs](https://docs.djangoproject.com/en/4.1/)

Needed installation

```bash
    source .env/bin/activate
    pip install django
```

Start the webserver

```bash
    python manage.py runserver 8000
```
