# API

- Current API version: `v1.0`
- developer: Peter Koban
- Framework: FastAPI [Docs](https://fastapi.tiangolo.com/)

Needed installation

```bash
    source .env/bin/activate
    pip install fastapi
    pip install "uvicorn[standard]"
```

Start the API

```bash
    uvicorn main:app --reload
```
