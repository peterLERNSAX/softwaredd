# Akteure und ihre Berechtigungen

| Akteur         | Berechtigung                      |
|----------------|-----------------------------------|
| Admin          | Vollzugriff                       |
| Außendienst    | Datenbank, Grundriss              |
| Innendienst    | Datenbank, Angebot                |
| Einkauf        | Datenbank, Angebot                |
| Verkauf        | Angebotsdatei                     |
| Server         | Angebot, Angebotsdatei, Datenbank |

+ Datenbankzugriff nur möglich, wenn Zugriff auf andere Abhängigkeiten gewährt wird
